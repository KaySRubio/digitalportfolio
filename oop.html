As a computer science student, I have programmed in Java and C++. Here's an example of a Java Program I wrote.


Challenge:

Code a program that analyzes a text file and produces new text randomly but in the style of the original text.


Potential Applications:

  • Artificial Intelligence
  • Predictive text suggestions in email or smart phones
  • Syntax-error checking in compilers

Solution:

I coded a program that uses a two-dimensional linked list structure to generate Markov models from a text file.


To generate text that is random but in the style of the original text, the program randomly chooses a word, then randomly chooses another word that follows that word anywhere in the text. Words that more frequently follow the first are more likely to be chosen. Then, the program chooses another word that follows the second word, and so on.

Skills Demonstrated:

  • Programmer-defined classes and methods
  • Defining relationships between classes
  • Complex data structures
  • Handling exceptions
  • Java library

Technical Details

Classes and Programming Steps

Steps the program follows:

  1. Obtain the filename from the user; catch any errors opening the file.
  2. Ask the user for the order of the Markov model
  3. Open the file, make all letters lowercase, remove all digits and symbols
  4. Initialize a KeyWordList
  5. Loop through the words in the file and initialize all unique words as a KeyWord objects
  6. Initialize each KeyWord's NextWordList
  7. Initialize words that follow the KeyWord anywhere in the file as NextWord objects in the NextWordList with frequencies
  8. Print the Markov Model for the user to view
  9. Ask the user how many words of random text they want the program to print
  10. Generate random text from the Markov Model
